# Icons Directory

## Current Status
Temporary placeholder icons are needed. The extension is ready to use but requires icon files.

## Required Icons
- icon-16.png (16x16 pixels)
- icon-48.png (48x48 pixels)
- icon-128.png (128x128 pixels)

## Quick Solution Options

### Option 1: Use icon.svg as reference
Open `icon.svg` in any browser and take screenshots at different sizes.

### Option 2: Online SVG to PNG converter
1. Go to https://cloudconvert.com/svg-to-png
2. Upload `icon.svg`
3. Convert to PNG at sizes: 16x16, 48x48, 128x128

### Option 3: Use any image editor
Create simple black and white icons with "LF" text or any generic design.

### Temporary Placeholder
You can use any 16x16, 48x48, and 128x128 PNG files as placeholders.
The extension will work without icons, they just won't display properly in the browser toolbar.

## Design Specification
- Background: White or transparent
- Foreground: Black
- Style: Minimalist
- Content: "LF" initials or chain link symbol

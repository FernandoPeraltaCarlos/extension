Link Finder

Searches visible page text in addition to link href attributes.

Allows partial matches (instead of exact) and includes relative paths (./, ../) and protocol-relative URLs (//example.com).

Matches the file name at the end of the link path (e.g. searching "/file.pdf" or "file.pdf" finds "domain.com/path/file.pdf"). Case-insensitive.
